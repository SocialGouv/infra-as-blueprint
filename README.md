# infra-as-blueprint

## getting started

clone repo

```sh
git clone git@github.com:SocialGouv/infra-as-blueprint.git
cd infra-as-blueprint
```

install

```sh
yarn
```

setup env

exec or put in `.env` file

```sh
export GITHUB_TOKEN=xxx
```

run

```sh
yarn start
```

## other commands

### help

```sh
yarn start --help
```


### dry-run

```sh
yarn start --dry-run
```

### debug

```sh
yarn dev
```
